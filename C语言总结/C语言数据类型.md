## 数据类型转换

### 自动类型转换

C语言中如果同一表达式中含有不同类型的变量，会自动进行类型转换

例如

> ```c
> int a = 2, double b = 2.0; 
> 
> printf("结果是：",a + b);
> ```
>
> 

> 结果是 4.0

结果被自动转换成了double类型

#### 自动类型转换规则

只有表示范围小的类型才能转换成表示范围大的类型

比如   可以由int类型转换成double类型

但不能自动从double类型转换成int类型
