**+** 代表拥有访问权限    **-**  代表没有访问权限

|            | Private | protected | public | default（包访问权限） |
| ---------- | ------- | --------- | ------ | --------------------- |
| 同一类内   | +       | +         | +      | +                     |
| 同一子类   | -       | +         | +      | -                     |
| 同一包内   | -       | +         | +      | +                     |
| 全局范围内 | -       | -         | +      | -                     |
